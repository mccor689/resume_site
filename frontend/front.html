Games By Ryan

Star Chase

This game is Star Chase. The objective of this game is to move the star character into the green squares while avoiding the red squares.

This game has persistent storage for the high score by writing to and reading a file.

Farkle

This game is Farkle. The goal of farkle is to be the first player to reach 10,000 points. The scoring is strange to first time players and there are some special cases.

• 1's are worth 100, 5's are worth 50

• Three of any one number in a single roll is worth the number times 100 (three 1's = 1,000)

• A straight (123456) on a single roll is worth 1,000

• If all dice score points then you get to reroll all dice adding on to the roll total

If your roll has no scoring dice, then you lose all accumulated points that turn, Farkle!

How Does This Work?

This project is built using two sides: a front and back.

Backend

The backend is a NodeJS file that can locally run on a computer. The backend uses the Cheerio and Axios modules scrape a website for NBA data and returns the matchups found on the website. The backend also uses the child_process module to create a child process when calling python files. Both games are written in python using the PyGames library, so the previous module is used to run the games.

Frontend

The frontend uses HTML and CSS to create this page seen and Javascript to call the backend and edit the page seen. The scripting makes calls to the backend to run the games and to pull data from the web scraper.
